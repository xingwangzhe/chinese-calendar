# Chinese Calendar GNOME Shell Extension

在 GNOME 时钟面板和日历卡片中显示中国农历日期、节气、传统节日和法定节假日信息。

## 功能特性

- 在顶部面板时钟旁显示农历日期
- 在顶部面板时钟旁显示节日信息
- 在日历弹窗中显示农历详情
- 在日历网格中显示农历日期
- 显示二十四节气
- 显示传统节日
- 显示法定节假日（休/班）
- 支持动态节日（母亲节、父亲节、感恩节）

## 安装方法

1. 下载扩展文件
2. 解压到 `~/.local/share/gnome-shell/extensions/` 目录
3. 重启 GNOME Shell（按 Alt+F2，输入 r 并回车）
4. 在 GNOME 扩展管理中启用扩展

## 设置选项

### 面板显示

- **面板显示农历**：在顶部面板时钟旁显示农历日期
- **面板显示节日**：在顶部面板时钟旁显示节日信息

### 法定假日

- **显示法定假日**：在日历中标记法定假日的"休"和"班"
- **数据源**：选择法定假日数据的来源（ShuYZ 镜像或 GitHub 官方源）
- **更新数据**：手动更新法定假日数据
